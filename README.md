## 製品概要
### 背景(製品開発のきっかけ、課題等）
色々なSNSをまたいで使用すると、切り替えが面倒だったり、連絡を見逃すことが多い。
SNSを一括して管理することによって、切り替えの手間を無くし、効率よく連絡をすることができるのではと考えた。

### 製品説明（具体的な製品の説明）
 - Twitter
 - Instagram→Facebook・Messenger
 - Discord
 - メール
